# Custom CSS Files

This directory contains all custom CSS files for UI customizations.

## Files

### login.css
Custom styling for the login page with:
- Modern gradient background
- Enhanced card design
- Improved input fields
- Eye icon for password visibility
- Responsive design

These CSS files are included in their respective templates via hooks configuration.
